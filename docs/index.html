    4. ユニットセル精密化

    「3. 回折点の表示」で選択した反射点それぞれについて、理論位置周辺を極座標(r, φ)でサンプリングし、動径方向ローレンツ×方位角方向ガウスの2Dモデルをフィットしてピーク中心を求めます。得られた d 値から最小二乗法で単位格子を精密化します。必要な点数と精密化するパラメータは「3. 回折点の表示」で選択した結晶系に従います。

    5. ローレンツ・偏光補正

    D, λ, p, cx, cy, 傾き補正角度は「2. 傾き補正」「3. 回折点の表示」で入力した値をそのまま使用します。繊維軸の傾きも「3. 回折点の表示」のφ(繊維傾斜)に既に組み込まれているため、ここでは改めて入力しません。強度抽出の最初のステップとして、円筒対称ローレンツ因子と偏光因子の逆数を生画像に掛けます。

    6. バックグラウンド除去

    A, 子午線方向は「5. ローレンツ・偏光補正」、D, λ, p, cx, cy, 傾き補正角度, 繊維軸の傾きは「2.」「3.」の入力値をそのまま使用します。ローレンツ・偏光補正済みの画像を極座標(φ, r)にリビニングし、方位角トレースごとにベイズ重み付きスプラインで背景を推定して差し引きます。数秒〜十数秒かかります。

    7. ピーク幅の算出

    「4. ユニットセル精密化」で選択・精密化した反射点それぞれについて、精密化で得たピーク位置(r0, φ0)を固定したまま、背景差し引き済みの極座標データに対して幅だけをフィットします。動径方向のローレンツ幅γrは全ての点を使ってwm/weq/cモデルをフィットし、方位角方向のガウス幅は l=0(赤道)の点のみを使って配向幅Bを求めます。「3. 回折点の表示」で選択する反射点には l=0 のものを少なくとも1点含めてください。「6. バックグラウンド除去」の結果を使うため、先に「6.」を実行してください。

    動径・方位角のフィット範囲は既定では「4. ユニットセル精密化」の探索窓と同じ値を使いますが、ここで変更できます。

    wm / weq / c (動径方向, 全点)

    「除外」にチェックした点を除いて「除外して再フィット」を押すと、その点を除いてwm/weq/cを再フィットします。

    配向幅 B (方位角方向, l=0反射のみ)

    8. ブロックフィット (強度抽出)

    「7. ピーク幅の算出」で得た wm/weq/c モデルと配向幅B(いずれもフィット済みの値をそのまま使用)を使い、動径方向をブロックに分けてNNLS(非負最小二乗法)で全反射の積分強度を同時に求めます。先に「6.」「7.」を実行してください。

    σI(強度誤差)の見積もりに使うパラメータです。相対誤差項は強反射のI/σ上限をおよそ1/relative_errorに制限します。較正倍率GooFは、SHELXL等で別途得た値がある場合にそれを入力するとσI全体に掛かります(既定1.0は較正なし)。

    面間隔でさらにマージ (対称等価だがhklが異なる反射)

    上の一覧に対し、同一l内で面間隔dが数学的にほぼ完全一致する(hklは異なる)反射同士をさらに加重平均します。許容誤差は浮動小数点誤差程度のごく小さい値が既定で、緩めすぎるとたまたま近いだけの別反射まで誤って平均してしまうので注意してください。
